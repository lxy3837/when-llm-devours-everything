[核心发现 - 2025-2026 AI+网络 热门趋势总结]
================================================================================

 >> 1. LLM + 网络是绝对的最大新热点 <<
   INFOCOM 2025 有 15+ 篇论文直接涉及 LLM
   方向包括: LLM边缘推理、KV Cache优化、联邦LLM微调(LoRA)、RAG、投机推理等
   2026 延续并深化: LLM语义缓存、多模态大模型(LMM)、端云协同LLM推理

 >> 2. 联邦学习持续火热，方向细化 <<
   2025年 23 篇联邦学习论文，覆盖:
     - 大模型联邦微调 (LoRA + FL) -> 交叉热点
     - 联邦遗忘 (Federated Unlearning) -> 新方向
     - 隐私攻击与防御 (梯度泄露/后门攻击) -> 持续关注
     - 激励机制、异构量化、个性化联邦等 -> 成熟方向深化

 >> 3. Transformer 全面渗透网络领域 <<
   Vision Transformer: 视频分析、IPv6地址生成
   Graph Transformer: 社交网络角色发现、流量工程
   2026 新方向: Spiking Transformer 训练优化

 >> 4. 扩散模型 (Diffusion) 快速崛起 <<
   应用场景: 语义通信、WiFi数据生成、网络功能部署、车辆检测、IPv6地址生成
   从CV领域向通信网络领域快速外溢

 >> 5. 强化学习是网络优化的稳定核心 <<
   MARL 用于数据中心、边缘计算、O-RAN、LPWAN
   新方向: 可解释DRL (SymbXRL)

 >> 6. MoE (Mixture-of-Experts) 崭露头角 <<
   从LLM架构向网络部署优化延伸
   2025: MoE理论 + serverless部署
   2026: MoE在线推理的专家合并 (Task-Aware Expert Merging)

 >> 7. 边缘AI推理成为基础设施级研究热点 <<
   模型从DNN扩展到LLM/LVM (Large Vision Model)
   关键技术: KV Cache、提前退出(Early Exit)、管道并行、量化调度、serverless推理

 >> 8. CNN 论文数量显著减少 <<
   仅出现在特定场景 (可编程数据平面、物理层)
   整体趋势: CNN -> Transformer -> LLM/MoE

 >> 9. RNN/LSTM 基本消失 <<
   被 Transformer 全面替代，仅在极少数时间序列预测中出现

 >> 10. RWKV 尚未出现在 INFOCOM <<
   作为较新的线性注意力架构，还未渗透入网络通信领域顶级会议
   可能是一个蓝海方向: RWKV的高效推理特性天然适合边缘设备部署

 >> 跨领域趋势总结 <<
   AI4Net (AI赋能网络) + Net4AI (网络支撑AI) 双轮驱动
   LLM的边缘化部署是 2025-2026 最显著的新增趋势
   安全+AI (后门攻击、投毒、推理攻击) 持续受关注
   绿色AI (能效优化) 在O-RAN和边缘场景受到重视
   Bandit/在线学习 是网络优化的底层理论支撑
